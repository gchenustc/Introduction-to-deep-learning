## 必要条件
执行源代码需要安装以下环境：
* Python 3.x
* NumPy
* Matplotlib

前进到各章节的文件夹，执行Python命令。

```
$ cd ch01
$ python man.py

$ cd ../ch05
$ python train_nueralnet.py
```
